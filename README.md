# YouTube Transcript

Script em bash para download de transcrições do YouTube.

## Pré-requisito(s)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- [yad](https://github.com/yt-dlp/yt-dlp) (para modo GUI)

## Modo de uso

```bash
./ytt https://www.youtube.com/watch?v=<ID>

Escolha uma opção:
p) Print
g) Gui
s) Save
q) Quit

Opção:
```
